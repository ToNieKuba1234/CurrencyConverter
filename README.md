# Przelicznik Walut – Currency Converter (PL)

**Autor:** Jakub Kasprzak

---

## Opis
Prosty, estetyczny i intuicyjny konwerter walut w Javie z interaktywnym interfejsem graficznym (GUI).  
Umożliwia szybkie przeliczanie między różnymi walutami oraz natychmiastowe aktualizacje przy zmianie wartości lub waluty.

---

## Funkcjonalności
- Wybór waluty źródłowej i docelowej.  
- Automatyczna aktualizacja przeliczenia po wpisaniu kwoty lub zmianie waluty.  
- Przyciski do zamiany walut miejscami (swap).    
- Estetyczny i intuicyjny interfejs użytkownika.  

---

## Wymagania
- Java 11 lub wyższa  
- Biblioteka Swing (wbudowana w JDK)  
- Klasa `RateAPI` obsługująca pobieranie kursów walut  

---

## Uruchamianie
1. Sklonuj repozytorium lub pobierz pliki projektu.  
2. Skonfiguruj klasę `RateAPI` do pobierania kursów walut.  
3. Skopiuj pliki `.java` do swojego IDE lub katalogu projektu.  
4. Uruchom klasę `Main.java`.

# Currency Converter – Przelicznik Walut (EN)

**Author:** Jakub Kasprzak

---

## Description
A simple, aesthetic, and intuitive currency converter in Java with an interactive graphical user interface (GUI).  
Allows quick conversions between different currencies and instant updates when changing the amount or currency.

---

## Features
- Selection of source and target currencies.  
- Automatic conversion update when entering an amount or changing the currency.  
- Buttons to swap currencies.  
- Aesthetic and intuitive user interface.  

---

## Requirements
- Java 11 or higher  
- Swing library (built into the JDK)  
- `RateAPI` class for fetching currency rates  

---

## Running
1. Clone the repository or download the project files.  
2. Configure the `RateAPI` class to fetch currency rates.  
3. Copy the `.java` files into your IDE or project directory.  
4. Run the `Main.java` class.
